# 🌤️ WeatherNow

A modern and responsive weather application that provides real-time weather information for any city using the OpenWeather API. Users can search for weather conditions worldwide or use their current location to receive accurate weather updates.

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Get weather using current location
- 🌡️ Real-time temperature, humidity, and wind speed
- 📅 5-Day weather forecast
- 🛰️ Live weather radar
- 🌫️ Air Quality Index (AQI)
- ☀️ UV Index information
- 💡 Weather-based health recommendations
- 🌙 Light/Dark mode toggle
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Geolocation API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/ritika913/Weather-site.git
```

2. Navigate to the project

```bash
cd Weather-site
```

3. Open `index.html` in your browser.

## 🔑 API Key

This project uses the **OpenWeather API**.

Create a free API key from:
https://openweathermap.org/api

Replace the API key in `script.js` with your own before deploying.
